Java-Wha

This project was originally inspired by another WhatsApp automation project I found on GitHub, which I later modified and customized for my own use cases.

I adapted the logic to allow scheduled messages based on date, time, and repetition intervals. Originally, I mostly made these changes as a joke/personal project to annoy my boyfriend a little when he stopped replying to me 😅.

Features I added/modified:

Send messages at a specific date and time
Repeat messages a custom number of times
Configure delays between messages
Personalized scheduling behavior

This repository is mainly for educational and experimental purposes.
